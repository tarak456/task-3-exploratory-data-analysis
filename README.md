# Task 3: Exploratory Data Analysis (EDA)

## Objective
To analyze and understand the dataset using visualizations and statistical summaries before applying machine learning.

## Dataset
- Iris Dataset

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Steps Performed
- Loaded and explored the dataset
- Analyzed numerical distributions using histograms
- Checked class distribution using count plots
- Detected outliers using box plots
- Studied feature relationships using correlation heatmap

## Key Insights
- Dataset is clean with no missing values
- Petal length and petal width are highly correlated
- Classes are well balanced
- Suitable for classification problems
